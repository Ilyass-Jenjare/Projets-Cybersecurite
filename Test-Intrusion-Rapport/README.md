# Test d'Intrusion (Pentest) et Rapport d'Attaque
**Rôle :** Analyste Sécurité Offensive

## Résumé du Projet
Simulation d'une attaque en boîte noire (Black Box) visant à cartographier la surface d'attaque, identifier les vulnérabilités et obtenir le plus haut niveau de privilèges sur un serveur cible, tout en documentant la chaîne d'attaque.

## Compétences Appliquées
* **Méthodologie PTES :** Respect strict du Penetration Testing Execution Standard.
* **Reconnaissance :** Énumération des services avec Nmap (HTTP, SSH, MySQL).
* **Exploitation :** Découverte et exploitation d'une porte dérobée (Bindshell port 1524) permettant un accès Root non authentifié.
* **Post-Exploitation :** Compromission d'une application web via Injection SQL et XSS.
