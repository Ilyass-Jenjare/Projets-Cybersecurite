# Investigation Forensique et Threat Hunting
**Rôle :** Analyste SOC / Incident Responder

## Résumé du Projet
Mise en place d'une architecture de réponse aux incidents (EDR) pour détecter, analyser et enquêter sur des compromissions avancées d'endpoints.

## Compétences Appliquées
* **Déploiement EDR :** Installation et configuration de Velociraptor.
* **Analyse d'Artefacts :** Détection de Memory Dumps et analyse post-nettoyage (MFT, USN Journal).
* **Threat Hunting :** Création et exécution de requêtes VQL (Velociraptor Query Language) pour identifier les indicateurs de compromission (IoC).
